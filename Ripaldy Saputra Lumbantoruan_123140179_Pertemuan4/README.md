# Program Pengelolaan Data Nilai Mahasiswa

Program Python untuk mengelola data nilai mahasiswa.

## Fitur

- **Tampilkan Data** - Melihat semua data mahasiswa dalam bentuk tabel
- **Tambah Data** - Menambah mahasiswa baru
- **Edit Data** - Mengubah data mahasiswa berdasarkan NIM
- **Hapus Data** - Menghapus data mahasiswa
- **Filter Grade** - Menampilkan mahasiswa berdasarkan grade (A/B/C/D/E)
- **Rata-Rata Kelas** - Menghitung rata-rata nilai seluruh mahasiswa
- **Nilai Tertinggi/Terendah** - Mencari mahasiswa dengan nilai tertinggi/terendah

## Sistem Penilaian

**Formula:** Nilai Akhir = (30% × UTS) + (40% × UAS) + (30% × Tugas)

**Grade:**

- A: ≥ 80
- B: 70-79
- C: 60-69
- D: 50-59
- E: < 50

## Cara Menjalankan

```bash
python program.py
```

Pilih menu sesuai kebutuhan (0-8).
